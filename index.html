推算の根拠となる理論・式

1. E型粘度計による粘度測定 生データからの推算

E型粘度計（コーンプレート型粘度計）において、回転数（N）とトルク指示値（TI）から、せん断速度（D）、せん断応力（\tau）、および見かけ粘度（\eta）を求める基本式です。

せん断速度 D \text{ [1/s]}

せん断応力 \tau \text{ [Pa]}

見かけ粘度 \eta \text{ [Pa·s]}

2. 流動曲線・粘度曲線のフィッティングモデル

流体のレオロジー挙動をモデル化するための各種構成方程式です。本システムでは実測データに対して非線形最小二乗法を用いてこれらのパラメータを同定します。

理想的な粘性流体 (Newtonian)

降伏応力を持つモデル (Bingham, Casson, Herschel-Bulkley)

べき乗則および全領域モデル (Power-law, Cross, Carreau)

3. 回帰評価指標とフィッティング理論

各モデルの当てはまりの良さ（適合度）を評価し、比較するための指標です。本システムでは外れ値の影響やモデルの絶対誤差を総合的に評価するため、以下の指標を算出しています。

二乗平均平方根誤差 (RMSE) と 平均絶対誤差 (MAE)

RMSEは大きな誤差を重くペナルティとして評価し、MAEは誤差の大きさを直感的に示します。※評価はせん断応力（\tau）基準で行われます。

決定係数 R^2

モデルが実測値のばらつきをどれくらい説明できているかを示す指標（最大1）。

4. 回帰手法・最適化アルゴリズム

実測データに対して各モデルの式を当てはめ、最適なパラメータを算出するための手法です。

線形最小二乗法 (Ordinary Least Squares: OLS)

Newtonian、Binghamなどの線形モデル、または対数変換等で線形化できるモデル（Casson、Power-law等）に対して用いられます。残差平方和が最小となるよう解析的にパラメータを求めます。

非線形回帰とパラメータ探索

Herschel-Bulkley、Cross、Carreau等の解析的に解けない非線形モデルについては、一部の非線形パラメータ（τ_y や λ など）をスイープ探索させながら線形最小二乗法を適用し、RMSEが最小となる組み合わせを探索・同定しています。

5. 記号・変数の定義一覧

  • N : 回転数 [rpm]
  • TI : トルク指示値 [%]
  • D : せん断速度 [1/s]
  • \tau : せん断応力 [Pa]
  • \eta : 見かけ粘度 [Pa·s]
  • \alpha : コーン角度 [rad]
  • R : コーン半径 [m]
  • T_{full} : フルスケールトルク [N·m]
  • \tau_y : 降伏応力 [Pa]
  • \eta_0 : ゼロせん断粘度 [Pa·s]
  • \eta_\infty : 無限せん断粘度 [Pa·s]
  • \eta_p : 塑性粘度 [Pa·s]
  • \eta_c : Casson粘度 [Pa·s]
  • K : 粘稠度指数
  • n : 流動挙動指数 (無次元)
  • m : Crossモデルの指数 (無次元)
  • \lambda : 緩和時間 [s]
  • \tau_{obs} : 実測せん断応力 [Pa]
  • \tau_{pred} : 予測せん断応力 [Pa]
  • \bar{y} : 実測値の平均
  • \hat{y} : モデルの予測値
  • \theta : 最適化対象のパラメータ集合
